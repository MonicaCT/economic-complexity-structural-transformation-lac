# Executive Summary

## Project purpose

This repository builds a reproducible economic-complexity workflow for Latin America, with a detailed application to Bolivia. It uses existing public trade-derived outputs to connect productive capabilities, export diversification, Product Space proximity and feasible structural-transformation opportunities.

## Analytical scope

| Dimension | Coverage |
|---|---:|
| Country-product-year observations | 6,497,429 |
| Countries | 242 |
| HS92 four-digit products | 1,243 |
| Period | 1995-2023 |
| Bolivia latest analytical year | 2023 |

## Core analytical products

- Country-year complexity and diversification panel.
- Product-year complexity panel.
- Product Space network diagnostics and visual edge subset.
- Bolivia opportunity-screening table.
- Bolivia strategic-bet review.
- Data-integrity and indicator-validation summaries.
- Paper, policy brief, dashboard screenshots and GitHub Pages gallery.

## Main findings

1. Bolivia remains in a low-complexity position in 2023 relative to the annual global distribution, with ECI reported as -1.236 in the validated repository outputs.
2. Bolivia's opportunity space includes a small set of highlighted strategic bets plus a larger universe of incremental and transformational candidates.
3. The Product Space is used as a screening framework: visual networks support communication, while analytical density and opportunity metrics rely on the full matrix.
4. Fixed-effects estimates in the paper are observational associations, not causal estimates.

## Recruiter relevance

The project demonstrates end-to-end analytical work: data validation, indicator engineering, reproducible reporting, network analytics, policy communication and dashboard-oriented presentation. It is strongest as evidence for data analyst, research data analyst, development data analyst and applied economics roles.

## Interpretation guardrails

- Opportunity rankings are screening tools, not investment recommendations.
- ECI and PCI are standardized within year and should be interpreted as relative positions.
- Product Space visualizations are reduced for readability; methodological interpretation relies on documented analytical outputs.
- No scientific content is changed by the portfolio flagship documentation layer.
