# GitHub Release Checklist

## Scientific

- [x] Indicators validated.
- [x] Claims reviewed for non-causal interpretation.
- [x] Limitations expanded.
- [x] Numerical consistency checked.
- [x] Simulated peer review completed.
- [x] Response to reviewer prepared.

## Technical

- [x] Tests pass.
- [x] Demo script runs from public samples.
- [x] Portability check completed.
- [x] Dashboard parses, loads data and exposes requested modules.
- [x] Paper HTML and PDF render.
- [x] File-size audit completed.

## Communication

- [x] README restructured.
- [x] Banner and dashboard preview created.
- [x] Final figure selection documented.
- [x] GitHub description and topics prepared.
- [x] Citation metadata reviewed.

## Privacy

- [x] No personal absolute paths in public files outside ignored local config.
- [x] No credentials or private tokens found by text scan.
- [x] Raw data and large caches excluded.
- [x] Temporary build files cleaned or ignored.

## Git

- [ ] Review `git status --short --ignored`.
- [ ] Confirm `.gitignore` before staging.
- [ ] Confirm branch and remote.
- [ ] Create first commit manually after human review.
- [ ] Push only after reviewing generated artifacts.
