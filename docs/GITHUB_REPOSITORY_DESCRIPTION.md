## LinkedIn announcement

I prepared a reproducible research portfolio project on economic complexity and structural transformation in Latin America, with Bolivia as the focal case. The repository reconstructs HS92 trade-based RCA, ECI, projected PCI, Product Space proximity, density and opportunity scores from local data; validates the main indicators; and distinguishes strict absolute opportunity thresholds from a relative Bolivia-specific screening universe. The project includes a working paper, policy brief, Shiny dashboard, public samples, figures, tests and audit reports. The key interpretation is cautious: product rankings are screening tools for further sector study, not automatic investment prescriptions.

## Research portfolio description

This project links Latin American economic complexity patterns to a Bolivia-focused diversification screen. It emphasizes reproducibility, validation and cautious policy interpretation: complexity metrics are used to organize feasibility questions, not to prescribe industrial policy automatically.
