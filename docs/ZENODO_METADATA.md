## Description

This release provides a reproducible research software repository for studying economic complexity, structural transformation and diversification opportunities in Latin America, with Bolivia as the focal applied case. The project uses country-product-year trade data to construct revealed comparative advantage, diversity, ubiquity, concentration, entropy, Economic Complexity Index, projected Product Complexity Index, Product Space proximity, density and opportunity metrics. The public release documents coverage of 6,497,429 country-product-year observations, 242 countries, 1,243 HS92 four-digit products and 1995-2023. It includes a Bolivia module that reports 2023 complexity and concentration indicators, classifies diversification candidates and treats product rankings as screening tools rather than automatic investment recommendations. The repository also contains a local Shiny dashboard, a working paper, a policy brief, figures, validation reports, public sample data and tests. Large raw source data and local processed caches are excluded from GitHub because of size and reproducibility boundaries; local configuration is separated from public files.
