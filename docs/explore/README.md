# Economic Complexity Interactive Explorer

This folder contains a static GitHub Pages explorer built only from existing final repository outputs. It does not execute the analytical pipeline and does not recalculate RCA, ECI, PCI, Product Space, econometric models, figures or tables.

## Views

- Executive Overview
- Country Explorer
- Product Complexity Explorer
- Product Space
- Bolivia Opportunity Lab
- Data Quality and Methodology

## Data sources

The CSV files in `data/` are lightweight presentation extracts from existing final outputs in `outputs/tables/csv/` and `outputs/networks/`.

## Publication

The explorer is designed for GitHub Pages from `main` and `/docs`:

`https://monicact.github.io/economic-complexity-structural-transformation-lac/explore/`
