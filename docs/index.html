Data and Analytical Workflow

Raw public trade data -> validation and harmonization -> country-product analytical panel -> RCA, ECI, PCI and network indicators -> opportunity screening -> figures, tables, dashboard and reports.

Coverage: 6,497,429 country-product-year observations, 242 countries, 1,243 HS92 four-digit products and 1995-2023. Large raw source folders are not distributed in GitHub.
